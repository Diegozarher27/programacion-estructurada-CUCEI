Actividad: Funciones en C

Descripción
En esta actividad se trabaja con el uso de funciones para modularizar programas en C. Se desarrollan ejemplos que van desde operaciones básicas como la suma de enteros hasta el recorrido de arreglos y matrices. También se analiza la diferencia entre distintos tipos de paso de parámetros a funciones.

Temas aplicados
- Operadores
- Arreglos
- Matrices
- Estructuras de control condicionales
- Estructuras de control cíclicas
- Funciones
- Punteros

Funciones utilizadas
En esta actividad se implementan funciones para:
- Sumar valores enteros
- Recorrer arreglos
- Manipular matrices
- Comparar diferentes formas de paso de parámetros

Compilación
gcc funcion_de_paso.c funcion_sumar.c funciones_arreglos.c funciones_matriz.c funciones_paso_parametro.c -o funciones

Ejecución
./funciones
