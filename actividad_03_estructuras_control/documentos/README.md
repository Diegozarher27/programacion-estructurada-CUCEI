Actividad: Estructuras de control en C

Descripción
Este conjunto de programas tiene como objetivo practicar y comprender el uso de estructuras de control en C, tanto condicionales como cíclicas. Se desarrollan ejemplos que permiten reforzar la lógica de programación mediante diferentes tipos de decisiones y ciclos.

Temas aplicados
- Operadores
- Estructuras de control condicionales
- Estructuras de control cíclicas
- Funciones
- Ciclos while, for y do-while
- Toma de decisiones con if, else if y else

Estructuras de control condicionales

Descripción
Programas que ejecutan estructuras de control condicionales y muestran su relación con los operadores.

Compilación
gcc edad.c else_if.c estructura_de_control_break.c impar.c par.c primo.c promedio.c validaciones.c -o estructuras_de_control_condicionales

Ejecución
./estructuras_de_control_condicionales

Estructuras de control cíclicas

Descripción
Programas que implementan estructuras de control cíclicas (while, for y do-while), simulando menús y reforzando el uso de ciclos en diferentes casos.

Compilación
gcc do_while.c for.c while.c -o estructuras_de_control_ciclicas

Ejecución
./estructuras_de_control_ciclicas

Compilación alternativa
gcc main.c
