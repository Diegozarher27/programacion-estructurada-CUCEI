Actividad: Arreglos y Matrices en C

Descripción
En esta actividad se trabaja con estructuras de datos como arreglos y matrices en C. Se desarrollan programas que permiten almacenar, recorrer y manipular colecciones de datos, así como simular problemas básicos utilizando estructuras bidimensionales.

Temas aplicados
- Operadores
- Arreglos
- Matrices
- Estructuras de control cíclicas
- Estructuras de control condicionales
- Librería stdlib.h

Arreglos

Descripción
Esta parte de la actividad se enfoca en el uso de arreglos para representar nombres completos, listas de nombres y el recorrido de estas listas mediante ciclos.

Compilación
gcc arreglos_direccion_mem.c arreglos_nombres.c arreglos_temperatura.c -o arreglos

Ejecución
./arreglos

Matrices

Descripción
Programa que simula el funcionamiento del juego "Battleship" utilizando matrices bidimensionales como estructura principal para representar el tablero y sus coordenadas.

Compilación
gcc matrices_coordenadas.c matrices.c -o matrices

Ejecución
./matrices
