Actividad: Punteros y Memoria dinámica en C

Descripción
En esta actividad se estudian los punteros y la memoria dinámica en C. Se realiza una demostración del uso de punteros, su relación con las direcciones de memoria y la asignación de memoria en tiempo de ejecución mediante malloc y free.

Temas aplicados
- Operadores
- Funciones
- Punteros
- Direcciones de memoria
- Estructuras de control cíclicas
- Arreglos
- Memoria dinámica (malloc y free)

Punteros

Descripción
Se realiza una demostración del uso de punteros y su relación con las direcciones de memoria de las variables a las que hacen referencia.

Compilación
gcc punteros_ejemplo.c punteros_funcion.c -o punteros

Ejecución
./punteros

Memoria dinámica

Descripción
Programa que reserva y libera memoria en tiempo de ejecución utilizando malloc, almacenando los datos en un arreglo unidimensional.

Compilación
gcc malloc.c -o malloc

Ejecución
./malloc
