 Actividad: Tipos de datos y Operadores Aritméticos en C
 Descripción
Este programa en C muestra el funcionamiento de los tipos de datos y los operadores aritméticos, además de imprimir la dirección de memoria de las variables.
 Temas aplicados
- Tipos de datos en C (int, float, double, char)
- Direcciones de memoria (&)
- Operadores aritméticos (+, -, *, /, %)
- Estructuras de control condicionales (if / else)
- Uso de printf

¿Qué hace el programa?
Tipos de datos
El programa declara variables de diferentes tipos y muestra:
- Su valor
- Su dirección de memoria
 Operadores aritméticos
Realiza operaciones básicas entre dos números:
- Suma
- Resta
- Multiplicación
- División
- Módulo
 Condicional
Compara dos valores y muestra cuál es mayor.
