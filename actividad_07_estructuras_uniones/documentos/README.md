Actividad: Struct y Union en C

Descripción
En esta actividad se implementan tipos de datos compuestos en C utilizando struct y union. Se estudia cómo struct permite agrupar diferentes variables bajo un mismo nombre, mientras que union permite compartir el mismo espacio de memoria entre distintos tipos de datos, optimizando el uso de memoria.

Temas aplicados
- Operadores
- Estructuras de control cíclicas
- Arreglos
- Estructuras de datos
- Struct
- Union

Struct y Union

Descripción
Programa que demuestra la creación y uso de tipos de datos compuestos. Se utiliza struct para agrupar múltiples variables relacionadas y union para observar el comportamiento de la memoria compartida entre variables.

Compilación
gcc recorrido_struct.c struct.c union.c -o structs_y_unions

Ejecución
./structs_y_unions
