Actividad: Manejo de cadenas en C

Descripción
En esta actividad se implementó un programa en C para trabajar con cadenas de caracteres utilizando funciones de la librería string.h. Se realizaron operaciones como copia, concatenación y comparación de cadenas.

El objetivo es comprender el manejo básico de cadenas y el uso de funciones estándar para su manipulación.

Objetivo
Aplicar funciones de la librería string.h para manipular cadenas y reforzar el uso de estructuras condicionales en C.

Temas aplicados
- Cadenas de caracteres (strings)
- Librería string.h
- Operaciones con cadenas:
  - Copia de cadenas
  - Concatenación
  - Comparación
- Estructuras condicionales (if / else)
- Operadores de comparación

Funciones utilizadas
- strcpy() → Copia una cadena en otra
- strcat() → Une dos cadenas
- strcmp() → Compara dos cadenas

Funcionamiento del programa
El programa realiza las siguientes acciones:

1. Declara varias cadenas de texto.
2. Copia una cadena dentro de otra usando strcpy.
3. Concatena cadenas usando strcat.
4. Compara dos cadenas usando strcmp.
5. Muestra resultados en pantalla dependiendo de las condiciones.

Compilación
gcc cadena.c -o cadenas

Ejecución
./cadenas

Resultado esperado
El programa imprime:
- Cadenas originales
- Resultado de la copia
- Resultado de la concatenación
- Resultado de la comparación entre cadenas
- Mensajes según condiciones evaluadas
